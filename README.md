## Installation

Provided step-by-step instructions on how to install and run this project locally.

```bash
# Clone the repository
git clone https://github.com/Sky-De/Expense_Tracker

# Change directory
cd client
cd server

# Add .env variables
check .env.example

# Install dependencies
npm install

# Run the project
npm run dev
```
